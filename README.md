# Digital-Assistant-for-Vehicle-Drivers
This was done as KUET CSE 3_2 System development project by 1607015 and 1607047. This is the desktop application, it also includes an android app. The prject can be proted to Raspberry pi and ultimately used in a car to-
1. Show and send the location to its owner,
2. detect if the driver is sleeping or not,
3. see the location and sleep status to android app.
